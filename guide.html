사용 가이드

라스베이거스 저유지비 리조트를 홈으로 사서 하와이를 공짜로 다니는 전략,
그 매물을 고르는 법을 단계별로 설명합니다.

시작하기

1

스크래퍼 실행 (선택)

실제 매물 데이터를 수집하려면 터미널에서 아래 명령을 실행합니다.
처음이거나 인터넷 수집이 막히면 샘플 데이터 옵션을 사용하세요.

cd C:\coding

python scraper.py --sample

→ listings.json 파일이 생성됩니다.

실제 스크래핑은 --sample 없이 실행. 실패 시 자동으로 샘플 데이터로 대체됩니다.
2

대시보드 열기

탐색기에서 dashboard.html을 더블클릭하거나, 터미널에서 실행합니다.

start dashboard.html
샘플 데이터는 대시보드에 이미 내장되어 있어 JSON 없이도 바로 사용 가능합니다.
3

JSON 데이터 불러오기 (선택)

스크래퍼로 수집한 실제 데이터를 보려면, 우측 상단 "JSON 불러오기" 버튼을 클릭해 listings.json을 선택합니다.

파일을 다시 선택하면 데이터가 즉시 갱신됩니다. 새로고침 불필요.

분석 지표 설명

MF/pt (핵심 지표)
유지비 ÷ 연간 포인트
포인트 1개당 연간 드는 유지비. 낮을수록 좋습니다. 이 숫자로 리조트 간 비교가 가능합니다.
목표: $0.12 이하
$/pt (구매 효율)
호가 ÷ 연환산 포인트
포인트 1개를 구매하는 비용. EOY 매물은 자동으로 포인트를 절반으로 환산합니다.
라스베이거스: $0.35 이하 적정
연간 총비용
(호가 ÷ 10년) + 유지비
구매가를 10년 상각한 값과 유지비를 합산한 실질 연간 비용입니다.
하와이 박수
보유 포인트 ÷ 7,000pt × 7박
Lagoon Tower 1BR Platinum(7,000pt/주) 기준으로 몇 박 이용 가능한지 추정합니다.
손익분기
$2,450 ÷ 연간 총비용
호텔 $350/박 × 7박($2,450/주) 대비 몇 년이면 본전이 나오는지 계산합니다.
3년 이내면 우수
ROFR 위험
Low / Med / High
가격이 너무 낮으면 힐튼이 선매권(Right of First Refusal)을 행사해 구매가 불발될 수 있습니다.
High = 힐튼이 낚아챌 가능성 높음
Value Score
0 ~ 100점 종합 평가
MF/pt (35%) + 구매가/pt (25%) + 포인트량 (15%) + 시즌 (15%) + Annual 여부 (10%)를 합산합니다.
80점 이상 = 매우 우수

MF/pt 색상 범례

녹색 — $0.12 이하 (우수, 목표 달성)
노란 — $0.12 ~ $0.15 (보통, 주의 필요)
빨간 — $0.15 초과 (비쌈, 비추천)

계산 예시

Elara 2BR Premier Platinum — 16,800pt Annual — 호가 $6,000

MF/pt$1,107 ÷ 16,800 = $0.066 ✓
구매가/pt$6,000 ÷ 16,800 = $0.357
연간 총비용($6,000 ÷ 10) + $1,107 = $1,707
하와이 박수16,800 ÷ 7,000 × 7 = 16.8박
손익분기$2,450 ÷ $1,707 = 1.4년
ROFR 위험Med ($/pt $0.357, 라스베이거스)
Value Score97점 / 100

필터 & 정렬 사용법

필

필터 패널

지역으로 Las Vegas / Hawaii / Orlando를 구분하고, 최대 MF/pt를 0.12로 설정하면 핵심 매물만 추립니다. 설정 후 "필터 적용"을 클릭하세요.

초보자 추천 세팅: 지역 = Las Vegas, 최대 MF/pt = 0.12, 사용방식 = Annual
정

컬럼 정렬

테이블 헤더를 클릭하면 해당 컬럼 기준으로 정렬됩니다. 한 번 더 클릭하면 역순. 기본 정렬은 Value Score 내림차순입니다.

시

하와이 시뮬레이터

사고자 하는 매물의 포인트를 입력하면 하와이 각 리조트·유닛에 몇 박이 가능한지 자동 계산합니다. Platinum / Gold 시즌별로 비교할 수 있습니다.

추천 투자 전략

LV 저유지비 메인 전략
  • Elara 1BR/2BR Premier Platinum 구매
  • MF/pt $0.066~0.091로 최저 수준
  • 포인트를 하와이로 옮겨 사용 (Club Exchange)
  • 라스베이거스는 홈 리조트 혜택만 활용
  • 호가 낮고 ROFR 위험 낮은 편
LV FREE 유지비만 인수
  • Boulevard 2BR, Flamingo 1BR 등 $0 매물
  • 초기 비용 없이 포인트만 확보
  • 유지비($839~$906)가 곧 포인트 비용
  • MF/pt $0.12~0.13으로 목표 근처
  • 점수는 낮아도 현금 부담 최소화
HI Kings Land 하와이 홈
  • 현지 홈 리조트 이점: 12개월 전 예약 가능
  • MF/pt $0.090으로 하와이 치고 양호
  • 호가 $8,000~$18,000으로 비교적 합리적
  • ROFR: $/pt $0.50 이상이어야 안전
  • EOY 매물로 초기 비용 낮추는 옵션
EOY 매물 전략
  • 격년(2년마다) 사용, 호가·MF 절반 수준
  • Elara 2BR Premier EOY: 호가 $1,200 선
  • 2개 구매(Even + Odd) = Annual과 동일 효과
  • 포인트 환산 시 Annual 기준으로 비교 필요
  • Value Score는 낮지만 현금 효율 좋음

주의사항

구매 전 반드시 확인하세요
  • ROFR High 매물은 힐튼이 선매권을 행사해 구매가 불발될 수 있습니다.
  • 유지비는 매년 약 5% 인상 추세입니다. 장기 계획 시 감안하세요.
  • EOY 매물은 포인트를 /2로 환산해 Annual과 동일 기준으로 비교해야 합니다.
  • 표시된 유지비가 누락된 매물은 DB 추정치이며 실제와 다를 수 있습니다.
  • $0 "FREE" 매물도 존재하나 유지비 부담은 그대로 인수합니다.
  • 이 도구의 데이터는 참고용이며 실제 구매 결정 전 전문가와 상담하세요.
